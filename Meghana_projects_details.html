School Admission Application

04/2022 - 06/2022

This application is created to drop an application request for XYZ school the user has to submit a new application form request by filling all the details like student name, father's name, mother's name, class of admission, phone number, address, email address etc. On submission of the form the data is first stored temporarily on students tables and the following cases can happen

Case 1 : If the admissions are closed for that academic year, then an alert is popped up immediately that the admissions are closed for that academic year.

Case 2 : If the admissions are open and seats are available for that class, then first an alert is popped up immediately that their information is stored and admission confirmation will be communicated shortly.

Case 2.1 : If the admission seat is approved by the school principal, then the user's data is automatically stored in the respective student's class table and the data from students table is deleted. The admission confirmation email is sent to the father's Email.

Case 2.2 : If the admission seat is not approved by the school principal, a regret email is sent to the father's Email stating the admission is rejected.

Case 3 : If the admissions are open and seats are not available for that class, then an alert is popped up immediately that the seats are full for that class.

Tools used : Service Now, flow designer, tables, forms, views, roles, scriptings

IOT based 'Light Intensity Monitoring System' for industrial purpose

09/2019 - 12/2019

Developed this device to measure the intensity of light. Very useful in places where the sunlight is very less but still wanted to plant trees at those places.

Working : The value of the resistance changes inversely with the amount of incident light. When the light intensity is high, the resistance is low and vice versa. By linking this device to the Bolt WI-FI module, information is collected at regular intervals of time. Thus we can know which area support plant growth.


Tools used : Bolt WI-FI module, Python, Light Dependent Resistor(LDR).

IOT based 'Temperature Monitoring System' for industrial purpose

06/2019 - 07/2019

This device has been developed to determine the temperature of the surrounding environment.

Working : Let's take the case of some medicine manufacturing unit where medicines require some standard temperature range, beyond which the medicines become toxic and leads to loss for the company. This device detects the temperature of the surroundings with the help of a sensor and checks if the temperature is within the specified range. If any abnormalities is detected then an immediate alert warning is sent to registered mobile numbers(of all the registered users). Thus, an immediate action is taken and risk of loss is minimized.


Tools used : Bolt WI-FI module, Python, temperature sensors
